# mediCal(Back-end)
  Backend server for the MediCal App

## Technologies
* Node.js
* Express
* MongoDB
* Bcrypt
## (User) Features
* Sign-in/Sign-up
* Add Appointment
* Update Appointment
* View all upcoming user appointments
## (Admin) Features
* View all
